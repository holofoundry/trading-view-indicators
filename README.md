# trading-view-indicators
A collection of Pine Script indicators for TradingView.

## Layout

Pine scripts live under [`pine/`](pine/), grouped by purpose so the repo stays easy to navigate as it grows.

- `pine/indicators/` - finished indicators organized by theme
- `pine/libraries/` - shared Pine helper code
- `pine/templates/` - starter files and reusable script skeletons
- `pine/experimental/` - work-in-progress ideas
- `pine/archive/` - retired or superseded scripts

See [`pine/README.md`](pine/README.md) for naming conventions and usage notes.
